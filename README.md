# 🎮Hangman
![Hangman](Images/hangman.png)
## Description
This game of Hangman was designed to be a fun game played by all. 
This program uses the Gnuci Disctionary to select a random word.
The player guesses letters to reveal the word before the player makes too many incorrect guesses.

## How to Play:
1. Click Start to begin the game.
2. A hidden word will appear as underscores.
3. Click any letter to guess if it is in the word.
4. If the letter is in the word it will appear in the correct place in the word.
5. If the letter is not in the word a man will begin to get hung.
6. Be careful because 6 wrong trys and you are out.
7. The games ends when:
   - you guess the word **Win!**
   - 6 wrong guesses **Lose**
   - The player may also press **Give up** and the game is over
  
## Features:
- Random words from Gnuci Dictionary
- Words are limited to 10 letters or fewer
- Visual Hangman appears using Graphics Paint
- Wrong guess counter
- Win and lose detection
- Give up button

## Technology Used:
- **C#**
- Windows Form
- Gnuci Dictionary
- Graphics drawing with Paint event
